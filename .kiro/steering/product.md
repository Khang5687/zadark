# ZaDark - Product Overview

ZaDark is a dark mode extension for Zalo PC (Vietnamese messaging app). It modifies the Zalo Electron desktop application to enable dark theme and privacy features.

## Platform
- **Windows** and **macOS** desktop applications

## Key Features
- Dark/Light/Auto theme modes
- Custom Google Fonts support
- Adjustable message font sizes
- Message translation
- Custom chat background images
- Privacy controls:
  - Hide latest messages in conversation list
  - Hide messages in chat thread
  - Hide conversation avatars and names
- Keyboard shortcuts for quick toggles

## Target Users
Vietnamese Zalo PC users who want dark mode and enhanced privacy controls.
